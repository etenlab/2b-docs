# Discussion API

## Description

Supports graphql apis for discussion box components, uses the discussions, posts, reactions tables.

## Installation

`npm install --legacy-peer-deps`

## Running the app

Set `env` vars:

```
PORT=8202
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=postgres
DB_PASSWORD=asdfasdf
DB_NAME=eil_db_1
```

`npm run start`

# watch mode

`npm run start:dev`

# production mode

`npm run start:prod`

## Test

`npm run test`

# e2e tests

`npm run test:e2e`

# test coverage

`npm run test:cov`
